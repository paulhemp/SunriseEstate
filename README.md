# Sunrise Estate Landing Page

This folder is ready for GitHub Pages.

## Files
- `index.html` — main landing page
- `assets/` — all images used by the page
- `.nojekyll` — tells GitHub Pages to serve the site as plain static files

## How to publish with GitHub Pages
1. Create a new public repository on GitHub.
2. Upload all files in this folder to the root of the repository.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Save.
6. Wait a minute or two. Your site will appear at:
   - `https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME/`

## Optional custom domain
If you want your own domain later, add it in **Settings → Pages** and point your DNS to GitHub Pages.
